# 🍙 おにぎり（Onigiri）のすべて

日本のソウルフードであり、シンプルながらも奥が深い「おにぎり（御握り）」。その歴史から定番の具材、美味しい作り方のコツまでをまとめました。

---

## 📌 おにぎりの基本形

おにぎりには地域や家庭によってさまざまな形があります。

*   **三角形**: 最も一般的。神の宿る山をかたどったという説もあります。
*   **俵（たわら）形**: 関西地方でなじみ深く、お弁当に詰めやすい形。
*   **丸形**: 手のひらで転がしやすく、焼きおにぎりにも最適。
*   **円盤（平丸）形**: 東北地方などでよく見られる、全体に味がなじみやすい形。

---

## 鮭・梅・昆布だけじゃない！ 定番＆人気具材

おにぎりの具材は無限大ですが、王道から進化系まで人気のラインナップをまとめました。

| 分類 | 具材名 | 特徴 |
| :--- | :--- | :--- |
| **不動の三種の神器** | 鮭（さけ） / 梅干し / 昆布 | 迷ったらこれ。保存性も高く、飽きない味。 |
| **コク旨・濃厚系** | ツナマヨ / 明太子マヨ | 子どもから大人まで大人気。コンビニの定番。 |
| **ガッツリ・肉系** | 肉巻き / 唐揚げ / 天むす | ボリューム満点。一つで満足感のある主食に。 |
| **変わり種・新定番** | 悪魔のおにぎり / チーズ天かす | 青のりや天かす、めんつゆを混ぜ込んだやみつき味。 |

---

## 💡 プロ直伝！ 美味しいおにぎりを作る3つのコツ

シンプルだからこそ、少しのこだわりで劇的に美味しくなります。

> ### 1. ご飯は「炊きたて」を、握りすぎない
> おにぎりは「握る」のではなく「形を整える」のが鉄則。ご飯の粒と粒の間に空気を含ませるように、3〜4回優しく包み込むように成形すると、口の中でほろりと解けます。
> 
> ### 2. 塩のタイミング
> 手水（てみず）に塩をつけるか、ご飯全体に軽く塩を混ぜ込んでおきます。目安はご飯1合に対して塩小さじ1/2弱。
> 
> ### 3. 海苔のタイミングはお好みで
> *   **パリパリ派**: 食べる直前に巻く。
> *   **しっとり派**: 握ってすぐに巻き、ご飯の水分で海苔を馴染ませる（磯の香りが引き立ちます）。

## 🌾 トリビア：おにぎり と おむすび の違い

諸説ありますが、一般的には以下のような違いがあると言われています。

*   **おむすび（御結び）**: 「むすび（結び）」の神に由来。神の力を授かるために**三角形**に握ったものが始まりとされる。
*   **おおにぎり（御握り）**: 「にぎりめし」が由来。形に決まりはなく、**握り固めたもの全般**を指す。

*※現在では、日本の地域やコンビニチェーンによって呼び方が異なるケースがほとんどです。*

## 📅 おにぎりの日
*   **6月18日**: 日本最古の「おにぎりの化石」が発見された石川県中能登町（旧鹿西町）が制定。
*   **1月17日**: 阪神・淡路大震災の日。ボランティアによるおにぎりの炊き出しの温かさを忘れないために制定（おむすびの日）。
